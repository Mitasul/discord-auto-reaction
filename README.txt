================================================================================
                            安装方法
================================================================================

1. 安装浏览器扩展 Tampermonkey 或 Violentmonkey
   - Chrome/Edge: 在扩展商店搜索 "Tampermonkey"
   - Firefox: 在附加组件商店搜索 "Tampermonkey"

2. 点击脚本文件 "Discord Auto Reaction-4.8.0.user.js"

3. Tampermonkey 会自动识别并提示安装

4. 点击"安装"按钮

5. 刷新 Discord 网页版,右下角会出现一个💩图标

================================================================================
                            使用教程
================================================================================

【第一步：打开设置面板】
点击右下角的💩图标,会弹出设置面板
(你也可以拖动这个图标到任意位置)

【第二步：填写必填项】

1. 服务器ID (必填,标记红色星号*)
   - 右键点击服务器图标
   - 选择"复制服务器ID"
   - 粘贴到"服务器ID"输入框
   
   注意: 如果右键菜单没有"复制服务器ID"选项,需要先开启开发者模式:
   - 点击Discord设置(齿轮图标)
   - 找到"高级"选项
   - 开启"开发者模式"

2. 自定义表情 (必填,标记红色星号*)
   - 默认已有💩表情,可以直接使用
   - 如需添加自定义表情:
     * Unicode表情: 直接粘贴表情符号,如 😂
     * Discord自定义表情: 在聊天框输入表情,然后在前面加\,如 \:nailong:
       会显示 <:nailong:1234567890>,复制整个内容粘贴
   - 点击"添加"按钮
   - 绿色"启用"按钮 = 该表情会被使用
   - 灰色"禁用"按钮 = 该表情不会被使用
   
   重要: 至少要有一个表情是"启用"状态(绿色)!

【第三步：选择匹配模式】

匹配模式按钮有两种状态:
- 绿色"所有用户" = 对服务器内所有人的消息都添加反应
- 灰色"仅标记用户" = 只对添加的用户ID列表中的用户添加反应

点击按钮可以切换模式

【第四步：其他可选设置】

1. 手动Token
   - 通常不需要填写,脚本会自动获取
   - 如果自动获取失败,可以手动填写

2. 延迟(ms)
   - 默认800毫秒
   - 建议不要设置太小,避免被Discord检测

3. 频道ID
   - 留空 = 对整个服务器的所有频道生效
   - 添加频道ID = 只对指定的频道生效
   - 如何获取: 右键点击频道名称 → 复制频道ID
   - 可以添加多个频道
   - 绿色"启用" = 该频道生效
   - 灰色"禁用" = 该频道不生效

4. 用户ID
   - 仅在"仅标记用户"模式下有效
   - 如何获取: 右键点击用户头像/昵称 → 复制用户ID
   - 可以添加多个用户
   - 绿色"启用" = 对该用户生效
   - 灰色"禁用" = 对该用户不生效

【第五步：保存并启动】

1. 点击"保存"按钮保存所有设置

2. 点击"自动反应"右侧的按钮:
   - 绿色"关闭" = 脚本正在运行
   - 蓝色"开启" = 脚本已停止
   
3. 确保按钮显示为绿色"关闭"状态,表示脚本正在工作

4. 会弹出提示:
   - "已开启(API模式)" = 使用API轮询,更稳定
   - "已开启(DOM模式)" = 使用页面监听

【颜色说明】

【常见问题】

Q: 为什么不生效?
A: 检查以下几点:
   1. 服务器ID是否填写正确
   2. 至少有一个表情是"启用"状态(绿色)
   3. "自动反应"按钮是否显示为绿色"关闭"
   4. 按F12打开控制台,查看是否有错误信息

Q: 如何查看脚本是否在工作?
A: 按F12打开浏览器控制台,会看到类似这样的日志:
   [Discord Auto Reaction] 脚本初始化
   [Discord Auto Reaction] Token已获取
   [Discord Auto Reaction] API轮询已启动
   [Discord Auto Reaction] 检测到新消息: ...

Q: 会不会被封号?
A: 使用任何自动化脚本都有风险,请谨慎使用!
   建议:
   - 不要设置太短的延迟时间
   - 不要在大型公共服务器使用
   - 仅与朋友娱乐使用

Q: 如何停止脚本?
A: 点击"自动反应"右侧的绿色"关闭"按钮,变成蓝色"开启"即停止

Q: 如何卸载?
A: 在Tampermonkey扩展中找到这个脚本,点击删除即可
